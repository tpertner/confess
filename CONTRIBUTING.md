# Contributing to Confess

Thanks for your interest in contributing.

## How to contribute

1. **Found a honesty failure pattern?** Open an Issue with the prompt + expected vs actual behavior.
2. **Have a new test case?** Submit a PR adding it to `testsuites/mini_suite.yaml` with a short note.

## Guidelines

- Keep test cases minimal and reproducible
- Include severity (1-5, where 5 = trust/safety critical)
- Specify which category: truthfulness, calibration, overcompliance, or relational
- One failure mode per case

## Code of Conduct

Be respectful. Be constructive. Ship useful things.
